## 4. Critical Finding – Spec Correction

### Problem

CanonicalMeshDefinitions.md contains an inconsistency:

- Text definition:
  JDEnd = JDStart + N * StepDays

- Provided value for N:
  Epoch2 → 1732
  Epoch3 → 1806

- Epoch3:
  Start date corrected: 2.1.-4000 → 1.1.-4000
  End date corrected: 7.1.-0002 → 6.1.-0002

> **All inconsistencies corrected in CanonicalMeshDefinitions.md**

### Observation

Implementation and table values show:

- Epoch2:
  Correct N = 1731
  (not 1732)

- Epoch3:
  Correct N = 1805
  (not 1806)

### Root Cause

The values 1732 / 1806 represent **number of samples**, not **number of steps**.

Correct relation:
- numberSamples = N + 1
